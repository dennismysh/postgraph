## Design Context

### Users
Solo creator analyzing and managing their Threads presence. Uses postgraph at a desk to understand engagement, plan content, and respond to replies. Future plans to open the platform to other creators for their own Threads accounts.

### Brand Personality
**Sharp, focused, efficient.** Like a well-tuned instrument — no fluff, everything earns its place. The interface should feel like it respects your time and attention.

### Aesthetic Direction
- **Visual tone:** Precision-crafted dark mode. Think Linear and Vercel Dashboard — clean, intentional, confident. Every element has a reason to exist.
- **References:** Linear (precise, dark-mode-done-right), Vercel Dashboard (minimal, fast, great typography)
- **Anti-references:** Hootsuite/Buffer (cluttered panels), generic SaaS dashboards (cookie-cutter metric cards), AI-generated dark mode (cyan/gradient/glassmorphism), enterprise software (dense/overwhelming)
- **Theme:** Dark mode only. Current palette of near-black backgrounds (#0a0a0a, #141414) with selective color accents for status/state.

### Design Principles
1. **Earn every pixel** — If an element doesn't help the user act or understand, remove it. No decoration for decoration's sake.
2. **State over style** — Color communicates meaning (status, urgency, success), not brand. Let the data speak.
3. **Density with clarity** — Show information efficiently without feeling cramped. Tight spacing is fine when hierarchy is clear.
4. **One thing at a time** — Each view has a primary purpose. Don't split attention across competing concerns.
5. **Speed is a feature** — Feel fast. Optimistic UI, instant feedback, no unnecessary loading spinners.
